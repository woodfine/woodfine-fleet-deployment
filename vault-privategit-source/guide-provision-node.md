# Provision Node Guide — vault-privategit-source

This guide covers provisioning the source code vault node that hosts private Git repositories for Woodfine-specific source code and configuration. Full provisioning steps will be documented when the cluster moves from scaffold-coded to active state.

## Prerequisites

- VM with SSH access enabled.
- Git server software (Gitea or equivalent) designated in the deployment plan.
- Network access from fleet nodes that will push or pull from this vault.
