# Provision Node Guide — route-network-admin

This guide covers provisioning the network administration node that holds the master WireGuard cryptographic keys and authoritative subnet assignments for the Woodfine fleet network. Full provisioning steps will be documented when the cluster moves from scaffold-coded to active state.

## Prerequisites

- VM or hardware with WireGuard installed.
- Static assignment in the mesh subnet (see `guide-mesh-orchestration.md`).
- SSH access to the target node.

## Steps

Detailed provisioning steps are pending. See `guide-mesh-orchestration.md` for key generation and subnet assignment.
