## I. TERMINAL ARCHITECTURE

Woodfine ledgers are secured via physical separation. **Type I Terminals** run on dedicated bare-metal hardware. **Type II Terminals** utilize the browser-based PointSav Console OS HUD on commercial machines for cross-cluster aggregation.
