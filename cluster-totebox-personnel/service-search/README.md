## I. OPERATIONAL MANDATE
This domain serves as the execution boundary for the Sovereign Inverted Index. It provides Woodfine operators with microsecond search capabilities across tens of thousands of corporate and personnel ledgers without relying on centralized, memory-heavy database servers (e.g., Elasticsearch).

## II. SOVEREIGN FRAMEWORK DECLARATION (DARP COMPLIANCE)
To enforce the Digital Asset Resolution Package (DARP) mandate, this search architecture is 100% portable. The index generated here is a static binary directory (`/search-index/`). If physical extraction is required for a compliance audit, this entire directory can be copied to an encrypted USB drive and queried on an air-gapped machine with zero database installation required.
