# 🚢 NODE-CONSOLE-OPERATOR
**Entity:** Woodfine Management Corp. (The Customer)
**Taxonomy:** Tier-3 Fleet Node (Unified Terminal)
**Status:** Provisioning

---

## I. OPERATIONAL MANDATE
This node represents the single, unified physical/virtual terminal utilized by Woodfine asset managers. It executes the `os-console` chassis and securely loads the `app-console-*` UI plugins (Keys, Content, Email, People) into a single pane of glass.

## II. SOVEREIGN FRAMEWORK DECLARATION
This terminal maintains zero localized corporate data. It strictly acts as a Secure Viewing Terminal, bridging encrypted connections to the underlying `cluster-totebox-*` execution vaults.
