# Provision Node Guide — fleet-infrastructure-onprem

This guide covers provisioning an on-premises hardware node into the Woodfine fleet network. Full provisioning steps will be documented when the cluster moves from scaffold-coded to active state. For LXC container setup, see `guide-lxc-network-admin.md`.

## Prerequisites

- Physical hardware with supported OS installation media.
- Network access to the WireGuard hub (see `fleet-infrastructure-cloud/guide-provision-relay.md`).
- SSH access to the target node after OS installation.
