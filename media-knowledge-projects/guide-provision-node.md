This guide covers provisioning a new node for the projects knowledge wiki media deployment. Full provisioning steps will be documented when the cluster moves from scaffold-coded to active state.

## Prerequisites

- VM with nginx and certbot installed.
- DNS A record for the projects wiki domain resolving to the VM's public IP.
