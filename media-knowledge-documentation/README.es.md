## I. MANDATO OPERATIVO

Este directorio actúa como puerta de enlace de despliegue para el wiki de documentación
de PointSav (`documentation.pointsav.com`). Contiene guías operativas para el
aprovisionamiento, despliegue y mantenimiento de la integración del sistema de diseño
y la configuración operativa del wiki.

## II. DECLARACIÓN DEL MARCO SOBERANO

Para garantizar el cumplimiento de SOC 3 y DARP, este nodo está estructurado para
operar en arquitectura aislada físicamente, de propiedad del Cliente. Aplica
estrictamente el Protocolo de Datos Soberanos, garantizando cero filtración hacia
analíticas de terceros o modelos de inteligencia externos.
